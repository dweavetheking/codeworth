# Contributing to Codeworth

Thank you for your interest in contributing!

## Getting Started

1. Fork the repo.
2. Clone it.
3. Run `npm install`.
4. Run `npm test`.

## Pull Requests

- Keep PRs small and focused.
- Ensure tests pass.
- Update documentation if needed.
